

# خطة: تحويل البكجات الجاهزة إلى بكجات قابلة للتخصيص الكامل

## الفكرة
تحويل صفحة البكجات الجاهزة من "تغيير التواريخ والمسافرين فقط" إلى نظام تخصيص كامل حيث يمكن للعميل تبديل الفندق، الطيران، السيارة، والجولات من بيانات فعلية مخزنة في قاعدة البيانات ومُدارة عبر لوحة التحكم.

## التعديلات المطلوبة

### 1. تحديث جدول `ready_packages` (Migration)
إضافة أعمدة ربط فعلية بالجداول الأخرى:
- `flight_offer_id` (UUID, FK → flight_offers) - الرحلة الافتراضية
- `car_rental_id` (UUID, FK → car_rentals) - السيارة الافتراضية
- `default_room_id` (UUID, FK → hotel_rooms) - الغرفة الافتراضية

هذه الأعمدة تحدد الاختيارات الافتراضية للبكج، لكن العميل يستطيع تغييرها.

### 2. إعادة بناء صفحة `ReadyPackages.tsx` - عرض التفاصيل
عند اختيار بكج، بدلاً من عرض ثابت، يظهر:

**قسم الفندق:**
- عرض الفندق الافتراضي للبكج
- زر "تغيير الفندق" يفتح قائمة الفنادق المتاحة من جدول `hotels` + `hotel_rooms`
- السعر يتحدث تلقائياً عند التغيير

**قسم الطيران:**
- عرض الرحلة الافتراضية
- زر "تغيير الرحلة" يفتح قائمة من `flight_offers`
- السعر يتحدث تلقائياً

**قسم السيارة:**
- عرض السيارة الافتراضية (إن وُجدت)
- زر "تغيير/إضافة سيارة" من `car_rentals`

**قسم الجولات:**
- عرض الأنشطة المتاحة من `tour_activities`
- إضافة/إزالة أنشطة مع تحديث السعر

### 3. منطق حساب السعر الجديد
بدلاً من `base_price_per_person` الثابت:
```
السعر = (سعر الطيران × المسافرين) + (سعر الفندق × الليالي) + (سعر السيارة × الليالي) + (سعر الأنشطة × المسافرين) + تأمين + فيزا
```
مع إبقاء `base_price_per_person` كسعر مرجعي/أدنى.

### 4. جلب البيانات
استخدام `tripBuilderService` الموجود مسبقاً لجلب:
- `getHotels()` + `getHotelRooms(hotelId)`
- `getFlightOffers()`
- `getCarRentals()`
- `getTourActivities()`

### 5. تحديث `handleSubmit`
حفظ الاختيارات الفعلية في `dynamic_packages`:
- `hotel_id`, `room_id`, `flight_offer_id`, `car_rental_id`
- `selected_activities` (JSON)
- حساب `subtotal`, `taxes`, `total_price` من الأسعار الفعلية

### 6. واجهة التخصيص (UI)
كل قسم يعرض:
- البطاقة المختارة حالياً (فندق/رحلة/سيارة) مع السعر
- زر "تغيير" يفتح Sheet/Dialog بالخيارات المتاحة
- عند الاختيار يُغلق ويُحدّث السعر فوراً
- تصميم متوافق مع الموبايل (Sheet من الأسفل)

### الملفات المتأثرة
- `supabase/migrations/` - migration جديد لإضافة أعمدة الربط
- `src/pages/ReadyPackages.tsx` - إعادة بناء كاملة لعرض التفاصيل
- `src/services/tripBuilderService.ts` - لا تغييرات (موجود مسبقاً)
