## 📝 ملاحظات التطوير

### متغيرات البيئة المطلوبة
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_DEEPSEEK_API_KEY=your_deepseek_api_key
```

### أوامر التشغيل
```bash
# تشغيل التطوير
npm run dev

# بناء الإنتاج
npm run build

# معاينة الإنتاج
npm run preview
```
