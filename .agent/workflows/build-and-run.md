---
description: Build and Run the Application
---

This workflow outlines the steps to build and run the Traveliun application.

1. Install dependencies (if not already installed)
```bash
npm install
```

2. Build the project for production
```bash
npm run build
```

3. Preview the production build (optional)
```bash
npm run preview
```

4. Start the development server
```bash
npm run dev
```
